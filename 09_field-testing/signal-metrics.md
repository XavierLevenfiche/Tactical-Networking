# Signal Measurement Metrics

Understanding network performance requires measuring signal characteristics.

---

## Key Metrics

### RSSI

Received Signal Strength Indicator

* measures signal power
* higher (closer to 0) is better

---

### MCS Rate

Modulation Coding Scheme

* determines data rate
* higher = better throughput

---

### Packet Loss

* percentage of lost packets
* indicates reliability

---

## Relevance

These metrics help evaluate:

* link quality
* network stability
* effective range
