# 802.11s Mesh Networking

802.11s is a wireless networking standard that enables mesh networking over WiFi.

Overview:
https://en.wikipedia.org/wiki/IEEE_802.11s

---

## Key Concepts

* nodes automatically discover peers
* traffic is forwarded across the mesh
* no central access point required

---

## Advantages

* native WiFi support
* higher bandwidth than LoRa
* IP-based networking

---

## Limitations

* shorter range than LoRa
* requires careful configuration

---

## Relevance

802.11s provides a foundation for building:

* high-bandwidth mesh networks
* IP-based communication systems
* distributed team networks
