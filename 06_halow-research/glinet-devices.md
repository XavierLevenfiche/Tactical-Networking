# GL.iNet Devices for Mesh Networking

GL.iNet produces compact routers that run OpenWRT.

Official site:
https://www.gl-inet.com

---

## Example Devices

* GL-AR300M16
* GL-MT300N (Mango)

These devices provide:

* low cost entry point
* portable form factor
* OpenWRT compatibility

---

## Advantages

* easy configuration
* USB power support
* community documentation

---

## Use Case in Project

These devices can function as:

* mesh nodes
* relay nodes
* testing platforms for routing protocols
