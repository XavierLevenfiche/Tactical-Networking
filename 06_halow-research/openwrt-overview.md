# OpenWRT Overview

OpenWRT is an open-source Linux-based operating system designed for embedded networking devices.

Official documentation:
https://openwrt.org

---

## Why OpenWRT

OpenWRT allows full control over:

* network interfaces
* routing protocols
* wireless configuration
* package management

This makes it ideal for building custom mesh networking nodes.

---

## Key Features

* package manager (opkg)
* support for mesh networking (802.11s)
* support for routing protocols (Babel, OLSR, BATMAN)
* configurable firewall and network stack

---

## Relevance to Project

OpenWRT enables the creation of:

* portable mesh nodes
* relay nodes
* TOC infrastructure

This marks the transition from **research → implementation**.
